JSON 模块

序列化的概念

把对象（变量）从内存中变成可存储或传输的过程称为序列化，序列化之后，就可以把序列化的内容写入磁盘或传输到其他的机器上，反过来，把变量内容从序列化的对象重新读到内存称为反序列化。

在 python 中直接打开文件写入集合会报错，需要将其类型转换为字符串。

pickle 模块与 json 模块的功能类似，但是 pickle 模块可处理函数等对象的写入，而 json 不行。
